# Phase v4.2.0 — 睡眠定时器 + 交叉淡入淡出（Crossfade）

**版本**: v4.2.0  
**状态**: complete  
**方向**: Flutter 客户端（播放体验延伸，依赖 v4.1.1 gapless 基础）

## 背景

v4.1.1 完成 `ConcatenatingAudioSource` 重构后，可以在队列层面做更细致的音频控制。
v4.2.0 在此基础上增加两项用户感知最强的播放体验功能：
- **睡眠定时器**：睡前听音乐常见需求，按时停止或当前曲目结束后停止
- **交叉淡入淡出**：曲目衔接时渐出/渐入，消除突兀的硬切换感

## 目标

- 睡眠定时器功能完整：两种模式（固定时长 / 曲目结束）、倒计时展示、取消
- Crossfade N 秒可配置（0–8s），Settings 「音频」section 直观 Slider
- `SleepTimerNotifier` 单元测试覆盖 4 种场景

## 依赖

- v4.1.1（`ConcatenatingAudioSource` gapless 基础，crossfade clip model 依赖此）

## 任务清单

### 睡眠定时器

- [x] **`SleepTimerNotifier`**（Riverpod Notifier）
  - `lib/src/audio/sleep_timer_notifier.dart`
  - 状态：`SleepTimerState { final Duration? remaining; final bool stopAfterTrack; final bool active; }`
  - `startFixed(Duration duration)` — 开始倒计时，每秒递减，到期调 `playerProvider.notifier.pause()`
  - `startAfterTrack()` — 监听 `playerProvider.processingState == completed` 后 pause
  - `cancel()` — 取消当前定时器，清除状态
  - 文件：`services/mobile/lib/src/audio/sleep_timer_notifier.dart`

- [x] **MiniPlayerBar 角标**
  - 睡眠定时器激活时，在 MiniPlayerBar 播放按钮右上角显示剩余时间（mm:ss，`stopAfterTrack` 时显示 ♪ 图标）
  - 文件：`services/mobile/lib/src/player/mini_player_bar.dart`

- [x] **FullPlayerScreen 控制栏图标**
  - 控制栏追加睡眠定时器按钮（月亮图标），激活时高亮（primary color），非激活时 onSurfaceVariant
  - 点击弹出 BottomSheet：固定时长选项（15 / 30 / 45 / 60 分钟）+ 「当前曲目结束」+ 「取消」（已激活时）
  - 文件：`services/mobile/lib/src/player/full_player_screen.dart`

- [x] **Settings 「音频」section 入口**
  - 追加 `ListTile`「睡眠定时器」，展示当前状态（未激活 / 剩余 mm:ss / 曲目结束模式）

  - 文件：`services/mobile/lib/src/settings/settings_screen.dart`

### 交叉淡入淡出（Crossfade）

- [x] **`CrossfadeNotifier`**（Riverpod Notifier）
  - `lib/src/audio/crossfade_notifier.dart`
  - 状态：`int crossfadeSeconds`（0 = 关闭，1–8 = 秒数）
  - `SharedPreferences` 持久化（key `audio.crossfade`）
  - 变更时通知 `AudioHandler` 更新 crossfade 设置
  - 文件：`services/mobile/lib/src/audio/crossfade_notifier.dart`

- [x] **AudioHandler crossfade 实现**
  - 读 `services/mobile/lib/src/player/audio_handler.dart`
  - 在 `currentIndexStream` 监听中，若 `crossfadeSeconds > 0`：
    - 在当前曲目剩余 `crossfadeSeconds` 秒时，开始当前曲渐出（`_player.setVolume` 线性降到 0）
    - 同步开始下一曲渐入（下一曲通过 `ConcatenatingAudioSource` 已预加载，`setVolume` 从 0 升到 1）
  - `crossfadeSeconds == 0` 时不做任何额外操作
  - 文件：`services/mobile/lib/src/player/audio_handler.dart`

- [x] **Settings 「音频」section Crossfade Slider**
  - `Slider` 值域 0–8，整数，0 显示「关闭」，1–8 显示「N 秒」
  - 文件：`services/mobile/lib/src/settings/settings_screen.dart`
