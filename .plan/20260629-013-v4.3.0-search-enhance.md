# Phase v4.3.0 — 搜索增强（Meilisearch 替换 PostgreSQL 全文搜索）

**版本**: v4.3.0  
**状态**: complete  
**方向**: 全栈（Go 服务端新搜索后端 + Docker Compose 新服务 + Flutter 搜索 UI 升级）

## 背景

v1.x 的 `GET /api/v1/catalog/search` 使用 PostgreSQL `to_tsvector` 全文搜索，存在以下局限：
- 无拼写容错（`battover` 不能匹配 `beethoven`）
- CJK 中文分词依赖 `pg_jieba` 扩展，部署复杂
- 无 autocomplete / 搜索建议

v4.3.0 引入 Meilisearch 作为搜索后端，保留 PG 全文搜索作降级路径，响应 schema 不变，Flutter 侧仅做 UI 增强。

## 目标

- Meilisearch 替换 PG 全文搜索，支持拼写容错和 CJK 分词
- Meilisearch 不可用时自动降级到 PG，对客户端透明
- Flutter SearchScreen 新增 autocomplete + Filter Tab
