# Phase v3.5.0 — 测试覆盖 + CI 完善

**版本**: v3.5.0  
**状态**: planned  
**方向**: 单元测试 / Widget 测试 / Integration 测试 + CI 三端构建

## 背景

v3.0.x 的 `flutter test` 仅有 8 个纯 Dart 测试（AuthState）；PlayerNotifier、FavoritesNotifier、HistoryNotifier、所有 Widget 均无覆盖，CI 也未验证 iOS 构建。

## 目标

- 单元测试 ≥ 30 cases
- Widget 测试 ≥ 10 cases  
- CI：APK + IPA（无签名）+ macOS 三端全绿
