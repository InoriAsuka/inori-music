# Phase v3.3.0 — 桌面平台增强 + 深度链接

**版本**: v3.3.0  
**状态**: done  
**方向**: 桌面平台功能 + 移动端深度链接

## 背景

v3.0.0 的桌面端（macOS/Windows/Linux）仅有自适应布局和键盘快捷键，缺少系统级集成（托盘、全局快捷键）。移动端也无法从外部链接打开指定曲目。

## 任务清单

### 桌面系统集成

- [x] 引入 `package:tray_manager ^0.2.3`（resolved: 0.2.4）
  - macOS/Windows/Linux：`TrayManager.instance.init('assets/images/tray_icon.png')`
  - 托盘菜单：Play/Pause / Next / Previous / Quit
  - `TrayListener.onTrayMenuItemClick` → 调 `playerProvider.notifier`
- [x] 引入 `package:hotkey_manager ^0.2.0`（仅 desktop）
  - 注册全局快捷键（`HotKeyManager.instance.register`）：
    - `Alt+Space` → togglePlayPause
    - `Alt+ArrowRight` → next
    - `Alt+ArrowLeft` → previous
  - `ConsumerStatefulWidget.dispose` 时 unregisterAll
- [x] `pubspec.yaml` desktop 依赖已添加（tray_manager, hotkey_manager）
- [x] `assets/images/tray_icon.png` 占位文件已创建
- [x] `lib/src/shared/desktop_integration.dart` 新建（DesktopIntegration class）
- [x] `lib/main.dart` → ConsumerStatefulWidget，initState 调 DesktopIntegration.init()

### 移动端深度链接

- [x] Android：`android/app/src/main/AndroidManifest.xml`
  - `<intent-filter>` 添加 `inori://` scheme（autoVerify=true）
  - 路径映射：tracks / albums / artists host
- [x] iOS：`ios/Runner/Info.plist`
  - `CFBundleURLSchemes: [inori]` 已添加
- [x] go_router 处理 deep link
  - 顶层 `/tracks/:id` route → `_DeepLinkTrackScreen`（initState 调 playTrack，完成后 go /player）
  - `AppRoutes.deepTrack/deepAlbum/deepArtist` 常量已添加
